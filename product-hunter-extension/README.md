# ProductHunter Extension

Plasmo browser extension project for ProductHunter.

## Current Scope

The extension contains popup and content-script code for in-page ProductHunter
comparison UI. It is present in the repository, but its production integration
and release flow are not fully documented yet.

## Development

```bash
npm run dev
```

Load the generated Chrome MV3 development build from:

```text
build/chrome-mv3-dev
```

## Build and Package

```bash
npm run build
npm run package
```
