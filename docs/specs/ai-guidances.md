# AI Guidance

This document provides AI agents guidance, guardrails, behaviors and strategies for the project. 

## General behavior guidelines
- Only research the whole project codebase when explicitly asked to do so. Otherwise, only research the relevant files and code snippets related to the question or task at hand.
- Only implement when explicitly asked to do so. Otherwise, only provide suggestions, recommendations, and guidance on how to implement the requested feature or fix the issue.
- Always ask for clarification if the question or task is ambiguous, incomplete, or unclear. Do not make assumptions or guesses, do not proceed to read the codebase or implement the solution until you have a clear understanding of the requirements and constraints.
- Only use plan mode when explicitly asked to do so or using plan mode. Otherwise, do not use plan mode. Always provide a clear and concise plan before implementing any solution, and ask for feedback and approval on the plan before proceeding with the implementation.
